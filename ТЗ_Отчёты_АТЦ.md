# ТЗ — Автоматизация отчётов АТЦ (1С Альфа-Авто → Excel → Битрикс24)

> Ред. 2 — учтены уточнения из обновлённой таблицы «Отчёты АТЦ — Финал для автоматизации (1).xlsx».
> Документ — рабочее задание для разработчика бэкенда и для 1С-разработчика.

## 1. Общая схема

```
1С Альфа-Авто  →  (REST / OData)  →  Бэкенд (Python, по расписанию)
                                        →  Заполняет Excel (5 АТЦ × 2 формы)
                                        →  Отправляет в Битрикс24
```

**Расписание:**
- **Ежедневно в 11:00** — отчёт за вчерашний день по каждому подразделению.
- **По пятницам в 11:00** — еженедельный отчёт за прошедшую неделю.

**Формы:** 5 направлений × 2 формы (Д — дневной, Н — недельный). Подразделений фактически 7 (две мойки).

**Легенда форм:** 🔵 Синий = из 1С автоматически · 🟡 Жёлтый = вводит руководитель вручную.

### 3.2. Состояния заказ-наряда — реальный список из 1С

Фактический справочник состояний ЗН из базы `test_moskva` (`Catalog_ВидыСостоянийЗаказНарядов`, 17 шт., проверено по OData). Использовать дословно:

`Заявка` · `Согласование` · `Согл гарантийного ремонта` · `Ожидание заезда на ремонт` · `Начать работу` · `В работе` · `Работы приостановлены` · `Ожидание автозапчастей клиента` · `Ожидание деталей` · `Ожидание оплаты` · `Опоздание начала работ` · `Опоздание окончания работ` · `Работы завершены` · `Выполнен` · `Закрыт` · `Отказ` · `Архив`

- ✔ **«ЗН ожидают ЗЧ» = считать ОБА статуса:** `Ожидание автозапчастей клиента` + `Ожидание деталей`.
- 🟧 **Уточнить:** «Активные ЗН» = все, кроме `Закрыт` и `Заявка`. Исключать ли также `Отказ`, `Архив`, `Выполнен`?
- ⚠️ Список снят с **тестовой** базы `test_moskva`. На боевой набор статусов может отличаться — перед запуском сверить с продакшеном.

### 3.3. Общие правила расчёта

- **Фактические нормочасы** — из табличной части заказ-наряда, столбец **«Норма времени»**.
- **Пересчёт месячного плана** в день — месячный план ÷ количество дней в месяце (в неделю — × 7).
- **Выручка без НДС** — регистр **«Доходы и расходы»**, статьи «Выручка по заказ-нарядам» и «Работы по заказ-нарядам».
- **Себестоимость без НДС** — регистр **«Доходы и расходы»**, статья «Себестоимость материалов».
- **Наценка на ЗЧ** — выручка − себестоимость (без НДС).
- **Ср. длительность ремонта** — по закрытым за неделю ЗН: разница дат «начат» и «закрыт».

### 6.2. Вопросы к 1С-разработчику, требующие ответа

1. 🟧 «ЗН без движения > 2 дней» — стандартный OData не отдаёт историю изменений. Есть ли реквизит «дата последнего изменения» или иной механизм?
2. 🟧 «Активные ЗН» — кроме «Закрыт» и «Заявка», исключать ли «Отказ», «Архив», «Выполнен»?

5. Регистр «Доходы и расходы» — регистр накопления? Доступен ли через стандартный OData, можно ли фильтровать по статье и подразделению?
